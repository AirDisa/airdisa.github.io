The best time to establish version control is beginning before the first version. Register with GitHub or refresh you familiarity with GitHub if you are already registered. GitHub is more than version control and it's where the developer community hangs out. GitHub is like the skateboard park for developers with company scouts looking for talent. Get GitHub going.
