New! Powered by [Jekyll][jekyll] templating. Jekyll supports code snippets:

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Disa')
#=> prints 'Hi, Disa' to STDOUT.
{% endhighlight %}
