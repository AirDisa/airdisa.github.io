Object-Oriented Programming

{{ site.time | date_to_string }} » Disa Johnson

To me, object-oriented programming and the MVC app framework has come to define modern software architecture on the Web. While aware of some of the alternates, the best reuse of code under the DRY principle is popular for good reason: organization. Since popularity also means more people will be programming in this style, it is important to write for others using the protocol they would come to expect and not something esoteric and foreign that would baffle them. Long gone are the protective programmer days, where people would hide their value in cryptic code. They still do that anyway haha.
